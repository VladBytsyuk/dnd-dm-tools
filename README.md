# DnD DM Tools для Obsidian

[![Tests](https://github.com/VladBytsyuk/dnd-dm-tools/actions/workflows/test.yml/badge.svg)](https://github.com/VladBytsyuk/dnd-dm-tools/actions/workflows/test.yml)

DnD DM Tools — плагин для [Obsidian](https://obsidian.md/) с русскоязычными инструментами и справочниками для Мастера Подземелий в Dungeons & Dragons 5-й редакции.

Текущая версия: **1.0.0-beta2**. Это предварительная версия для локального тестирования. Минимальная версия Obsidian — **0.15.0**.

## Помощник ДМа

Все инструменты открываются в едином представлении **«Помощник ДМа»**:

- глобальный поиск по разделам и объектам;
- вкладки с изменением порядка и переносом между областями;
- одна или две вертикальные рабочие области с настраиваемым размером;
- сохранение открытых вкладок, активной области и компоновки между запусками.

![Помощник ДМа с вкладками и двумя рабочими областями](/img/two_tiles.png)

![Глобальный поиск по разделам и объектам](/img/omni_search.png)

## Возможности

Плагин включает 13 инструментов:

| Инструмент | Возможности |
| --- | --- |
| Бестиарий | Просмотр, поиск, создание и редактирование статблоков существ |
| Книга заклинаний | Справочник, поиск, создание и редактирование заклинаний |
| Ширма | Быстрый доступ к правилам и справочным материалам |
| Арсенал | Справочник оружия |
| Броня | Справочник доспехов |
| Экипировка | Справочник обычного снаряжения |
| Магические предметы | Справочник магических предметов |
| Предыстории | Справочник предысторий персонажей |
| Черты | Справочник черт |
| Расы | Расы и подрасы |
| Классы | Классы, архетипы и их описания |
| Листы персонажей | Импорт, создание, редактирование и локальное хранение листов |
| Трекер инициативы | Порядок ходов, состояния, ресурсы и ячейки заклинаний участников |

Большие справочники загружаются постранично. Поддерживаемые сущности и бои можно отображать непосредственно в заметках с помощью Markdown-блоков плагина.

## Установка из исходного кода

Для сборки требуются Node.js 18 или 20, npm и хранилище Obsidian для тестирования.

```bash
git clone https://github.com/VladBytsyuk/dnd-dm-tools.git
cd dnd-dm-tools
npm ci
npm run release
```

После сборки:

1. Скопируйте каталог `.release/dnd-dm-tools` в `<ваше-хранилище>/.obsidian/plugins/dnd-dm-tools`.
2. Перезапустите Obsidian или перезагрузите плагины.
3. Откройте **Настройки → Сторонние плагины** и включите **DnD 5e DM tools**.

Каталог установленного плагина должен содержать:

```text
dnd-dm-tools/
├── main.js
├── manifest.json
├── sql-wasm.wasm
└── styles.css
```

## Использование

1. Нажмите кнопку **«Помощник ДМа»** на левой ленте Obsidian.
2. Выберите нужный раздел на панели инструментов.
3. Используйте строку поиска для поиска по доступным справочникам.
4. Переключите компоновку, чтобы открыть вторую рабочую область.
5. Перетаскивайте вкладки или используйте их кнопки управления для изменения порядка и переноса.

Состояние Помощника сохраняется автоматически.

### Ссылки на сущности

В заметках можно вручную создавать ссылки на сущности плагина. Используйте обычный Markdown-формат с протоколом `dnd:`:

```markdown
[Fireball](dnd:/spells/fireball)
[Гоблин](dnd:/bestiary/goblin)
```

Текст в квадратных скобках — это алиас ссылки. При клике ссылка открывает соответствующую сущность в **«Помощнике ДМа»**. Подробнее: [ссылки плагина](docs/plugin-links.md).
