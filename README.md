# DnD DM Tools - Плагин для Obsidian

[![Tests](https://github.com/VladBytsyuk/obsidian-dnd-statblock/actions/workflows/test.yml/badge.svg)](https://github.com/VladBytsyuk/obsidian-dnd-statblock/actions/workflows/test.yml)

Этот плагин для Obsidian предоставляет набор инструментов для Мастеров Подземелий (ДМ) в Dungeons & Dragons 5-й редакции. Он разработан, чтобы помочь вам эффективно управлять игровыми элементами прямо в вашей базе знаний Obsidian.

## Возможности

### Бестиарий
Управляйте существами, их характеристиками и описаниями. Легко добавляйте, просматривайте и редактируйте статблоки монстров.

<Скриншот>

### Книга заклинаний
Ваш личный справочник по заклинаниям. Создавайте, редактируйте и быстро находите необходимые заклинания.

<Скриншот>

### Трекер инициативы
Следите за ходом боя с помощью удобного трекера инициативы. Управляйте порядком ходов, состоянием существ и эффектами.

<Скриншот>

### Ширма мастера
Боковая панель для быстрого доступа к основным правилам, таблицам и другой важной информации по D&D 5e.

<Скриншот>

## Установка (WIP)

1.  **Откройте Obsidian:** Запустите ваше приложение Obsidian.
2.  **Перейдите в настройки:** Нажмите на иконку шестеренки в левом нижнем углу, чтобы открыть "Настройки".
3.  **Плагины сообщества:** В боковом меню выберите "Сторонние плагины".
4.  **Отключите безопасный режим:** Убедитесь, что "Безопасный режим" отключен.
5.  **Просмотрите плагины:** Нажмите кнопку "Просмотреть", чтобы открыть список доступных плагинов.
6.  **Найдите "DnD DM Tools":** В строке поиска введите "DnD DM Tools" и найдите плагин в списке.
7.  **Установите:** Нажмите кнопку "Установить".
8.  **Включите плагин:** После установки вернитесь в "Сторонние плагины" и включите "DnD DM Tools" с помощью переключателя.

## Использование

После установки и включения плагина, вы найдете новые команды и виджеты в Obsidian.

*   **Доступ к инструментам:** Откройте командную палитру (Ctrl/Cmd + P) и начните вводить "DnD DM Tools", чтобы увидеть доступные команды.
*   **Использование боковой панели:** Некоторые функции, такие как "Ширма мастера" и "Трекер инициативы", могут быть доступны через боковые панели Obsidian.
